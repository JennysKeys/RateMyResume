# RateMyResume

## Instructions For Running Locally:
1) Open project
2) Run `npm i`
3) Run `npm run start`
    - The console should say `Listening at: http://0.0.0.0:8080/`
4) Open a new tab on your browser
5) Go to `http://0.0.0.0:8080/`
